# crud-ordem-servico-node
Teste técnico para a vaga de programador júnior, onde foi desenvolvido um CRUD de ordens de serviço em Node.js e React.js

# Executando o projeto

Para executar o projeto, realize as seguintes intruções:

- Baixe o projeto, nele existem as pastas de frontend e backend
- Tenha o postgresql instalado e executando, crie um schema chamado ```orders```
- Execute o comando ```npm install``` para baixar todas as dependências de ambos os projetos
- Para rodar o 'backend', execute o comando ```npx tsx index.js``` em algum editor de código com o projeto aberto
- Para rodar o 'frontend', execute o comando ```npm run dev``` em algum editor de código com o projeto aberto

Com ambos os projetos rodando, basta acessar http://localhost:5173/ e todas as funcionalidades estarão disponíveis! ✅
